Centro de ayuda

En esta sección encontrarás información básica para utilizar Virtual Academy, tanto si eres estudiante como si formas parte del personal.

Guía rápida para estudiantes

  • Inicia sesión con tus credenciales asignadas.
  • Accede a Información del estudiante para ver tus asignaturas y calificaciones.
  • Descarga tu informe académico en PDF si lo necesitas.

Guía rápida para secretaría

  • Gestiona usuarios y sus roles.
  • Administra asignaturas.
  • Asigna estudiantes a materias y calificaciones.

Guía rápida para profesorado

  • Consulta la lista de alumnos y sus asignaturas.
  • Actualiza calificaciones cuando lo requiera el curso.
  • Revisa que la información del alumno sea correcta.

Preguntas frecuentes

•

No puedo iniciar sesión

Contacta con secretaría para restablecer tu contraseña.

•

Mis datos no son correctos

Reporta el error a secretaría para corregirlos.

•

No veo mis asignaturas

Puede que aún no hayan sido asignadas. Contacta con tu profesor o secretaría.
